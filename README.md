# MyIpTool

实时获取当前设备网络 IP 地址的单页html

- 打开网页实时获取当前设备网络 IP 地址，支持多线路，包括中国移动、中国联通、中国电信、海外线路，以及访问谷歌等特殊网站。
- 通过加载对应网站上的小图片进行测试网站联通性及加载延迟。
- 支持输入 ip 查询相关信息。
